# Proyecto de Business Intelligence

## Descripción

Este proyecto de Business Intelligence (BI) está diseñado para proporcionar soluciones de análisis y visualización de datos que ayuden a las organizaciones a tomar decisiones basadas en datos. Con una arquitectura que incluye un frontend desarrollado con Vue.js y un backend desarrollado con Java, este proyecto integra diversas fuentes de datos, procesa información y presenta resultados visuales de manera interactiva.

**Características del proyecto**:
- **Frontend**: Interfaz de usuario con Vue.js para la visualización de datos interactivos.
- **Backend**: API RESTful desarrollada con Java para manejar las consultas y operaciones de datos.
- **Visualización de datos**: Uso de herramientas de visualización interactivas en el frontend.

## Arquitectura del Proyecto

Este repositorio contiene dos proyectos principales:

1. **Frontend**: Implementado en Vue.js para la visualización interactiva de datos.
2. **Backend**: Implementado en Java (Spring Boot) para gestionar las operaciones y lógica del servidor, además de la conexión a la base de datos.

## Tecnologías utilizadas

### Frontend:
- **Vue.js**: Framework progresivo para construir interfaces de usuario.
- **Vuex**: Gestión del estado de la aplicación.
- **Vue Router**: Manejo de las rutas y navegación.
- **Fetch**: Para hacer peticiones HTTP al backend.

### Backend:
- **Java**: Lenguaje de programación para la lógica del servidor.
- **Spring Boot**: Framework para construir aplicaciones web y APIs RESTful.
- **JPA / Hibernate**: Para la persistencia de datos en bases de datos.
- **MySQL**: Base de datos para almacenar los datos procesados (según configuración).
